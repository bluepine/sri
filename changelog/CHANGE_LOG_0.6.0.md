# ChangeLog 0.6.0


## Breaking Changes

Project `groupID` and `artifactID` are changed.

groupID :  `com.github.chandu0101.sri`  to `com.github.chandu0101`

artifactID : `web` to `sri-web` , `mobile` to `sri-mobile`
